# Package Checklist

Use this checklist before delivery:

- Source code, tests, docs, and lockfiles are present.
- Local-only artifacts are removed: virtual environments, dependency folders, caches, logs, coverage, build outputs.
- Secrets are absent or replaced by documented placeholders.
- `.env.example` or equivalent configuration guidance exists.
- README includes setup, run, test, and troubleshooting sections.
- Test/build command has been run after cleanup, or the blocker is documented.
- Large generated data is intentionally kept or intentionally removed.
